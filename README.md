# String

A string builder implemented using memmove for crafting.

# Installation

- Install the library

`em list`

`em install`

`em build lib`

## Usage

- Include in a file

`touch test.c`

```c
#include "export/String.h"
int main(void) {
}
```

- Compile

`clang test.c export/*.*o $(find ../libs -name "*.*o" | xargs ls -d)`
